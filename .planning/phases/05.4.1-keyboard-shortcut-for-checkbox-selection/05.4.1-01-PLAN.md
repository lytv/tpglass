   - **Space**: Toggle selection mode on/off. When entering mode, set `keyboardFocusedIndex = 0`. When exiting, reset to -1.
   - **Escape**: If in selection mode, exit selection mode (keep selection). Track previous mode state to handle double-Escape clearing.

   - **ArrowDown**: Move focus down one row (if index < messages.length - 1), prevent default scroll behavior
   - **Space or Enter**: Toggle checkbox selection for current focused row, update selectionAnchor
   - **Shift+Arrow**: Range selection - select all rows between selectionAnchor and new position

   - `tabindex="0"` to make container focusable
   - `@keydown="${this._handleKeyboardNavigation}"` to handle keyboard events

   - Change condition from `this.isHoveringList` to `this.isHoveringList || this.isSelectionMode`
