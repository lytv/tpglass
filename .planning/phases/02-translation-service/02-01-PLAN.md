  <action>
Create translationService.js in src/features/common/services/ following these patterns:

1. Import dependencies:
   - const crypto = require('crypto') (built-in for SHA-256 hashing)
   - const { createLLM } = require('../ai/factory')
   - const modelStateService = require('./modelStateService')

2. Create TranslationService class with:
   - this.cache = new Map() for in-memory caching
   - this.defaultModel = 'gpt-4o-mini' for cost-effective translation

3. Implement _generateCacheKey(text, targetLanguage, sourceLanguage):
   - Normalize text: text.trim().replace(/\s+/g, ' ')
   - Generate SHA-256 hash: crypto.createHash('sha256').update(`${source}:${targetLanguage}:${normalized}`).digest('hex')
   - Handle sourceLanguage = null as 'auto'

4. Implement translate(text, targetLanguage, sourceLanguage = null):
   - Check cache first using _generateCacheKey, return cached result if hit
   - Get API key via modelStateService.getCurrentModelInfo('llm')
   - If no API key, return {success: false, translatedText: text, error: 'API key not configured'}
   - Build system prompt: "You are a professional translator. Translate the following text from [sourceLanguage] to [targetLanguage]. Respond with ONLY the translation, no explanations."
   - For auto-detect (sourceLanguage null): "Detect the source language and translate to [targetLanguage]. Respond with ONLY the translation. If source is unclear, respond with original text."
   - Use createLLM with modelInfo.provider, apiKey, model, temperature: 0.3, maxTokens: 4096
   - Call llm.chat([{role: 'system', content: prompt}, {role: 'user', content: text}])
   - On success: cache result, return {success: true, translatedText: result.content, error: null}
   - On error: return {success: false, translatedText: text, error: error.message}

5. Implement clearCache() method: this.cache.clear()

6. Export singleton: module.exports = new TranslationService()

Key requirements per SERV-01, SERV-02, SERV-03:
- Auto-detect source language when sourceLanguage is null
- Return original text with error indicator on failure
- Cache results using SHA-256 hash of normalized text + languages
