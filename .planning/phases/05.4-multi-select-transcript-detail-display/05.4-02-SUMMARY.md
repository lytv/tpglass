# Phase 05.4 Plan 02: Bulk Action Handlers with Combined Text

## Summary

Modified the action handlers (Summarize, Custom Prompt) to work with combined text when multiple transcripts are selected. When multiple selected, all texts are combined and the LLM operation runs once.

## Implementation Details

### _getCombinedText() Helper Method
- Returns single transcript text if only one selected (including translation if enabled)
- Combines multiple transcripts with `\n\n---\n\n` separator
- Includes translation for each transcript if translation is enabled
- Filters out empty messages

### _handleSummarize() Modifications
- Checks if multi-select mode is active (`selectedTranscriptIds.length > 1`)
- Uses `_getCombinedText()` for multi-select
- Falls back to single transcript text for single select
- Maintains original behavior when no selection active (uses `expandedTranslation`)

### _handleRunCustomPrompt() Modifications
- Same pattern as `_handleSummarize()`
- Supports multi-select, single-select, and original behavior

### _handleTranslationClick() Modifications
- When 2+ transcripts selected + click = opens multi-select modal
- When 1 selected + click = switches to single view (unchanged)
- When 0 selected + click = original behavior (opens single transcript modal)
