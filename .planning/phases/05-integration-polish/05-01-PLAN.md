#### Task 2: Integrate Translation with Listen View

**Files:**
- `pickleglass_web/app/activity/details/page.tsx` - Add translation integration

**Action:**

1. **Import translationApi at top of file:**
   ```typescript
   import { translationApi } from '@/utils/translationApi';
   ```

2. **Add state for translation:**
   ```typescript
   const [translatedText, setTranslatedText] = useState<Record<string, string>>({});
   const [translationEnabled, setTranslationEnabled] = useState(false);
   const [targetLanguage, setTargetLanguage] = useState('en');
   const [isTranslating, setIsTranslating] = useState(false);
   ```

3. **Add useEffect to load translation settings on mount:**
   ```typescript
   useEffect(() => {
     const loadSettings = async () => {
       const settings = await translationApi.getSettings();
       if (settings) {
         setTranslationEnabled(settings.enabled);
         setTargetLanguage(settings.language);
       }
     };
     loadSettings();
   }, []);
   ```

4. **Add translation fetch function:**
   ```typescript
   const fetchTranslation = async (transcriptText: string) => {
     if (!translationEnabled || !transcriptText) return;

     setIsTranslating(true);
     try {
       const result = await translationApi.translate(transcriptText, targetLanguage);
       if (result.success) {
         // Store translated text by transcript ID
         setTranslatedText((prev) => ({ ...prev, [transcriptId]: result.translatedText }));
       }
     } catch (error) {
       console.error('Translation error:', error);
     } finally {
       setIsTranslating(false);
     }
   };
   ```

5. **Trigger translation on transcript load (when enabled):**
   - Call fetchTranslation when sessionDetails.transcripts is loaded and translationEnabled is true
   - Pass each transcript's text to the translation function

6. **Listen for settings changes:**
   ```typescript
   useEffect(() => {
     const handleSettingsChange = async (settings: any) => {
       setTranslationEnabled(settings.enabled);
       setTargetLanguage(settings.language);

       // Re-translate if enabled and transcript exists
       if (settings.enabled && sessionDetails?.transcripts) {
         for (const item of sessionDetails.transcripts) {
           await fetchTranslation(item.text);
         }
       }
     };

     // Listen for translation settings changes
     if (window.api) {
       window.api.on('translation-settings-updated', handleSettingsChange);
     }

     return () => {
       if (window.api) {
         window.api.off('translation-settings-updated', handleSettingsChange);
       }
     };
   }, [sessionDetails]);
   ```

7. **Update the translated text display to use translatedText state:**
   - Replace the mock translation display with actual translatedText[ item.id ]

**Verification:**
- Translation settings loaded on page mount
- Translation triggers when enabled and transcript available
- Settings changes in Settings view reflect immediately in Listen view
- Toggle still controls display mode only (per user decision)

**Done:** Translation auto-triggers when enabled, settings sync in real-time
