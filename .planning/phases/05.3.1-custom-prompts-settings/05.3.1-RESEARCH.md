## Summary

This phase implements Custom Prompts Settings - allowing users to create, edit, and delete custom AI prompts in Settings, and execute them from the Custom Prompt modal in SttView. The feature reuses existing infrastructure (preset system, SummarizeService, IPC handlers) to minimize new code while providing a complete user experience.

**Primary recommendation:** Extend existing preset CRUD infrastructure to support custom prompts, add a new UI section in SettingsView below Translation Language settings, and implement custom prompt execution in SttView modal using an extended SummarizeService.

### Pattern 1: Service Extension for LLM Execution
**What:** Extend existing SummarizeService with new method for custom prompts
**When to use:** Need to execute LLM with user-defined prompts
**Example:**
```javascript
// Extend SummarizeService with runCustomPrompt method
async runCustomPrompt({ prompt, transcriptText }) {
    const modelInfo = await modelStateService.getCurrentModelInfo('llm');
    if (!modelInfo || !modelInfo.apiKey) {
        return { success: false, error: 'API key not configured' };
    }

    const llm = createLLM(modelInfo.provider, {
        apiKey: modelInfo.apiKey,
        model: modelInfo.model,
        temperature: 0.5,
        maxTokens: 2048,
    });

    const messages = [
        { role: 'system', content: 'You are a helpful assistant.' },
        { role: 'user', content: `${prompt}\n\nTranscript:\n${transcriptText}` }
    ];

    const completion = await llm.chat(messages);
    return { success: true, result: completion.content };
}
```

## Common Pitfalls

### Pitfall 1: Default Prompts Not Inserted on First Launch
**What goes wrong:** Users don't see default prompts after fresh install
**Why it happens:** Database schema exists but default prompts aren't seeded
**How to avoid:** Add default prompt seeding in databaseInitializer or migrationService
**Warning signs:** Preset dropdown shows empty when it should have 5 defaults

### Pitfall 2: Default Presets Are Editable
**What goes wrong:** Users can modify or delete default prompts
**Why it happens:** Update/delete queries don't check is_default = 0
**How to avoid:** SQLite repository already checks `AND is_default = 0` in update/delete queries
**Warning signs:** Default prompts appearing in editable list

### Pitfall 3: Long-running LLM Calls Block UI
**What goes wrong:** Modal freezes while waiting for LLM response
**Why it happens:** Synchronous IPC call blocks renderer
**How to avoid:** Return immediately from IPC handler, use events to stream results back
**Warning signs:** Modal unresponsive during execution

### Pitfall 4: Prompt Overflow in Small Transcript
**What goes wrong:** Empty or very short transcript leads to poor results
**Why it happens:** No minimum text length validation
**How to avoid:** Check transcript length before execution, show appropriate message
**Warning signs:** "No transcript available" error when modal opened with empty transcript

## Code Examples

### Extending SummarizeService
```javascript
// Source: Based on summarizeService.js structure
// File: src/features/common/services/summarizeService.js

async runCustomPrompt({ promptId, prompt, transcriptText }) {
    if (!transcriptText || !transcriptText.trim()) {
        return { success: false, error: 'No transcript text provided' };
    }

    if (!prompt || !prompt.trim()) {
        return { success: false, error: 'No prompt provided' };
    }

    try {
        const modelInfo = await modelStateService.getCurrentModelInfo('llm');
        if (!modelInfo || !modelInfo.apiKey) {
            return { success: false, error: 'API key not configured' };
        }

        const llm = createLLM(modelInfo.provider, {
            apiKey: modelInfo.apiKey,
            model: modelInfo.model,
            temperature: 0.5,
            maxTokens: 2048,
        });

        const messages = [
            { role: 'system', content: 'You are a helpful assistant that analyzes transcripts.' },
            { role: 'user', content: `${prompt}\n\nTranscript:\n${transcriptText}` }
        ];

        const completion = await llm.chat(messages);
        return { success: true, result: completion.content };
    } catch (error) {
        console.error('[SummarizeService] Custom prompt error:', error.message);
        return { success: false, error: error.message };
    }
}
```
