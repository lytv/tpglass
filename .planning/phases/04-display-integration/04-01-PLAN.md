#### Task 1: Add Translation Toggle Component

**Action:**
Add a toggle switch at the top of the transcript section in `/pickleglass_web/app/activity/details/page.tsx` that allows users to switch between original and translated text display.

**Verification:**
- Toggle appears at top of transcript section
- Toggle has "Translated" label with icon
- Default state is OFF (shows original text)
- Toggle state changes on click

**Implementation details:**
- Use React useState for toggle state
- Read initial state from localStorage if available
- Add Lucide React translation icon

```tsx
// Add to imports
import { Languages } from 'lucide-react';

// Add toggle state
const [showTranslation, setShowTranslation] = useState(false);

// Add toggle UI before transcript section
<div className="flex items-center gap-2 mb-4">
  <Languages className="h-4 w-4 text-gray-500" />
  <span className="text-sm text-gray-600">Translated</span>
  <button
    onClick={() => setShowTranslation(!showTranslation)}
    className={`relative inline-flex h-6 w-11 items-center rounded-full transition-colors ${
      showTranslation ? 'bg-blue-600' : 'bg-gray-200'
    }`}
  >
    <span className={`inline-block h-4 w-4 transform rounded-full bg-white transition-transform ${
      showTranslation ? 'translate-x-6' : 'translate-x-1'
    }`} />
  </button>
</div>
```

**Verification:**

- Translated text displays in right panel (or bottom on mobile)

      <div className="space-y-3">
        {/* Map through translated transcripts - same structure as original */}
        {sessionDetails.transcripts.map((item) => (
          <p key={`translated-${item.id}`} className="text-gray-700">
            <span className="font-semibold capitalize">{item.speaker}: </span>

      <p className="text-gray-500 italic">Enable translation to see translated text</p>

  </div>

#### Task 4: Add Copy Translation Button

**Action:**
Add a copy button next to each translated transcript entry that copies the translated text to clipboard using Clipboard API.

**Verification:**
- Copy icon appears next to each translated transcript entry
- Clicking copies text to clipboard
- Success feedback (optional: toast or brief visual change)

**Implementation details:**
- Import Copy icon from lucide-react
- Add click handler using navigator.clipboard.writeText

```tsx
import { Copy, Check } from 'lucide-react';

// Add copy state tracking (optional - for showing success feedback)
const [copiedId, setCopiedId] = useState<string | null>(null);

// Add copy handler
const handleCopyTranslation = async (text: string, id: string) => {
  try {
    await navigator.clipboard.writeText(text);
    setCopiedId(id);
    setTimeout(() => setCopiedId(null), 2000); // Reset after 2 seconds
  } catch (err) {
    console.error('Failed to copy text:', err);
  }
};

// Add copy button in translated panel
<div key={`translated-${item.id}`} className="flex items-start justify-between group">
  <p className="text-gray-700 flex-1">
    <span className="font-semibold capitalize">{item.speaker}: </span>
    {/* Translated text */}
  </p>
  <button
    onClick={() => handleCopyTranslation(item.text, item.id)}
    className="ml-2 p-1 text-gray-400 hover:text-gray-600 opacity-0 group-hover:opacity-100 transition-opacity"
    title="Copy translation"
  >
    {copiedId === item.id ? (
      <Check className="h-4 w-4 text-green-500" />
    ) : (
      <Copy className="h-4 w-4" />
    )}
  </button>
</div>
```
